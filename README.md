# Certifications: PL-200 Power Platform Functional Consultant

Study materials, walkthroughs, and practice projects for the **Microsoft PL-200: Power Platform Functional Consultant** certification.

PL-200 sits between PL-900 (fundamentals) and PL-400 (developer). It focuses on configuring solutions, working with Dataverse, building model-driven and canvas apps, automating with Power Automate, and supporting users.

## Who this is for

- Anyone preparing for the PL-200 exam
- Functional consultants new to Power Platform
- Makers wanting to move beyond the basics
- Citizen developers stepping up to more advanced configuration

## What this repo will cover

- Configuring Microsoft Dataverse
- Designing and building model-driven apps
- Designing and building canvas apps
- Power Automate flows: cloud, desktop, business process
- Integrating with other Microsoft services
- Solutions, environments, and basic ALM
- Practice projects and study notes
